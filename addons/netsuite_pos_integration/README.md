# NetSuite POS Integration Module

Enterprise-grade integration between Odoo Point of Sale and NetSuite ERP.

## Features

- ✅ Real-time and batch synchronization
- ✅ Configurable sync modes without code changes
- ✅ Queue-based background processing
- ✅ Automatic retry mechanism
- ✅ Comprehensive audit logging
- ✅ Manual sync controls
- ✅ Role-based permissions
- ✅ Mock NetSuite server for testing

## Quick Setup

1. Install the module
2. Navigate to: NetSuite → Configuration
3. Configure API URL and sync settings
4. Test connection
5. Start syncing!

## Sync Modes

**Real-time**: Syncs immediately on order confirmation
**Batch**: Accumulates orders and syncs in scheduled batches

## Manual Controls

From POS Order:
- Sync to NetSuite
- Force Resync
- View Sync Logs
- View Queue Status
